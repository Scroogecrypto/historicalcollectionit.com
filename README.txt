Historical Collection company website - stable video build.

All files are intentionally placed in the root folder. No assets/images/videos folders.

Video handling:
- home-background.mp4 optimized to ~1.4MB with faststart.
- contact-background.mp4 optimized to ~6MB with faststart.
- home-poster.jpg and contact-poster.jpg are real frames from videos.
- video tags use poster + preload="auto" + playsinline to prevent logo fallback delay.

Upload all files directly to GitHub repository root.
